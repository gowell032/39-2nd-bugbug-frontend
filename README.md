# BugBug


## 프로젝트 취지 및 소개
* 개발 기간 : 2022.11.28~ 12.09
* 프로젝트 사이트: 텀블벅
* 세상에 없던 크라우드펀딩 서비스
* 창작자와 후원자가 소통하는 서비스
  * 창작자가 상품을 올리면 후원자는 상품에 후원 할수있습니다.
  * 상품은 일정 기간 목표한 금액을 채우면 후원자는 상품을 받을수 있습니다.
